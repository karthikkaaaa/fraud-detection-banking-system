# TrustShield Bank - Fraud Detection System

A Django-based banking system with fraud detection and complaint management.

## Features

- User Registration & Login
- Make Transactions
- Fraud Risk Scoring
- Flagged Transaction Alerts
- Email Notification System
- Bank Admin Dashboard
- Customer Complaint System
- Staff Management
- Search Functionality

## Technologies Used

- Python
- Django
- SQLite
- HTML & CSS

## How to Run

1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run migrations:
   python manage.py migrate
4. Run server:
   python manage.py runserver
